* How to create users in NCI Report Writer.
  * Login into UPT as an administrator and as ncireportwriter application:
    * For local builds use: http://localhost:19280/upt
      * Login ID: [Administrator ID]
      * Password: [Administrator Password]
      * Application Name: ncireportwriter
  * Create an admin user:
      * Select "User" tab link.
      * Select "Create a New User" link.
      * Fill in new user details form.
        * Select Add button.
      * Select Association Groups button.
      * Select "admin" value from Available Groups list.
        * Select Assign button.
        * Select Update Association button.
      * Select Assign PG & Roles button.
        * Select "admin_pg" value in Available Protection Groups list.
        * Select Assign button.
        * Select "admin_role" value in Available Roles list.
        * Select Assign button.
        * Select Update Association button.
      * Select Associated PE & Privileges button.
        * Verify "admin-pe" in part of the list.
        * Select Back button.
      * Select Associated PG & Roles button.
        * Verify "admin-pg" in part of the list.
        * Select Back button.
  * Create an regular user:
      * Select "User" tab link.
      * Select "Create a New User" link.
      * Fill in new user details form.
        * Select Add button.
      * Select Association Groups button.
      * Select "non-admin" value from Available Groups list.
        * Select Assign button.
        * Select Update Association button.
      * Select Assign PG & Roles button.
        * Select "non-admin_pg" value in Available Protection Groups list.
        * Select Assign button.
        * Select "non-admin_role" value in Available Roles list.
        * Select Assign button.
        * Select Update Association button.
      * Select Associated PE & Privileges button.
        * Verify "admin-pe" in part of the list.
        * Select Back button.
      * Select Associated PG & Roles button.
        * Verify "admin-pg" in part of the list.
        * Select Back button.
